## Usage Examples

### For Blog Posts
```tsx
import {SocialHead, createSocialPreview, blogSocialPreviewTemplate} from '@openclint/docusaurus-shared/ui';

<SocialHead socialMeta={createSocialPreview(config, blogSocialPreviewTemplate)({
  title: "My Blog Post",
  description: "An amazing blog post about...",
  author: "Author Name",
  tags: ["tag1", "tag2"]
})} />
```

### For Documentation
Just use frontmatter - it's automatically enhanced:
```yaml
---
title: My Documentation
description: Comprehensive guide to...
author: Technical Writer
image: /img/custom-preview.png
---
```
