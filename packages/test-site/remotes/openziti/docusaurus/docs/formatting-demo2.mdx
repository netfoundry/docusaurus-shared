:::

## Lists with code

- Create an identity with `ziti edge create identity`
- Enroll with `ziti edge enroll token.jwt`
- Connect:
  ```bash
  ziti-edge-tunnel run -i identity.json
  ```

## Code inside table

| Command | Description |
|---------|-------------|
| `ziti edge list services` | List all services |
| `ziti edge create config` | Create a config |
| `ziti edge delete service` | Delete a service |
