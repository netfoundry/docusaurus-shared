# Images

captions. Uses `static/img/logo.svg` as the placeholder.

## Light / dark theme variants

import ThemedImage from '@theme/ThemedImage';
import useBaseUrl from '@docusaurus/useBaseUrl';

<ThemedImage
  alt="Themed logo"
  sources={{
    light: useBaseUrl('/img/logo.svg'),
    dark: useBaseUrl('/img/logo.svg'),
  }}
  width={200}
/>

Both variants currently point at the same SVG -- swap one for a dark-mode-specific
asset once it exists to verify the theme switch.
